## ✨ Key Features

| Feature | Description |
|---|---|
| 🧠 **Neural Loader** | Cinematic boot-up sequence that sets the premium tone before revealing the core interface |
| 🔥 **Flame Typography** | Hero name ignites with a custom CSS fire animation on load |
| 💫 **Canvas Particle Engine** | Custom vanilla JS neural network visualization — interconnected nodes, zero dependencies |
| 💻 **Interactive Terminal Hub** | `$ connect kartikeya` — A live terminal with animated connection steps and clickable contact cards |
| 🌐 **Globe Hologram** | Atmospheric holographic globe rendered in the terminal background |
| 📜 **Scroll Reveal System** | Lightweight `IntersectionObserver` scroll animations across every section |
| 📊 **Journey Timeline** | Interactive horizontal timeline showcasing milestones and growth |
| 📱 **Fully Responsive** | Precision media queries from 4K down to 320px small-screen devices |

## 🏃 How to Run Locally

No `npm install`. No build step. No config.

```bash

# Open: http://localhost:8000

# Option 2 — Double click
# Just open index.html in any modern browser
```
